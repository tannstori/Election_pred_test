🔧 Manual Turnout Override Guide
✓ Supported Override Scopes:
• municipality - Apply turnout to entire municipality (e.g., "Torshavn", "Klaksvik")
• site - Apply turnout to specific polling location (e.g., site IDs like "1001", "1002")
✓ CSV Format (config/live_turnout_overrides.csv):
scope,name,turnout_votes,turnout_percent
• Fill turnout_votes if you have absolute count
• Fill turnout_percent if you have percentage only
• Leave blank if not known (system infers from available data)
Example Entries:
municipality,Klaksvik,2887, (Klaksvik: 2,887 votes, % inferred)
municipality,Runavik,,77.2 (Runavik: 77.2% turnout, votes inferred)
site,1045,,68.5 (Site 1045: 68.5% turnout)
